# Contributing

欢迎围绕“论文工作流方法论”提交改进。

## 适合提交的内容

- 工作流说明改进
- 计划模板优化
- 本地 skill 设计经验
- 公开安全的示例文档
- 路由、handoff、审校流程的改进建议

## 不接受的内容

- 论文正文成品
- 原始实验数据
- 第三方论文 PDF 原文
- 私有截图
- 带本机绝对路径的未清理文档
- 未确认授权的第三方 prompt / skill 原文大段复制

## 提交前请自查

1. 不要带入私有盘符路径。
2. 不要带入个人隐私信息。
3. 不要把本仓库变成论文正文仓库。
4. 如果引用第三方 skill / prompt，请写清来源。
5. 如果是本地化改造，请优先说明“设计思路”和“边界”，而不是只贴结果文本。
