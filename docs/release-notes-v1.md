## 关于原始计划文件中的绝对路径

`workflow-assets/plans/` 下的计划文档保留了原始本地绝对路径，例如：

- `G:\bishe\JLU-CCST-Thesis\...`

这是刻意保留的，因为这些计划文件同时也是这次论文写作的原始过程资产。

但如果你想复用这套方法：

- 不要直接照抄路径
- 应把这些路径替换成你自己的项目路径

也就是说：

- 这些计划文件保留的是“结构和执行逻辑”
- 不是“必须逐字照搬的文件位置”
